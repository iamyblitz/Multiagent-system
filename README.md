# Мультиагентная система для вычисления среднего

## Описание

Система из 15 агентов, организованных в минимальное остовное дерево (MST).
Агенты вычисляют среднее значение распределенно через агрегацию снизу вверх.

## Стоимости операций

- `msg = 0.1` - сообщение между агентами
- `msg_c = 1000` - сообщение центру
- `операция (+,-,/) = 0.01` - арифметическая операция
- `mem = 0.1` - хранение числа в памяти

## Структура проекта

```
Homework1/
├── config.py          # Константы стоимости операций
├── graph_utils.py     # Создание графа и построение MST
├── agent.py           # Класс TreeAgent с поведением агрегации
├── main.py            # Точка входа в программу
├── requirements.txt   # Зависимости
└── venv/             # Виртуальное окружение
```

## Запуск

### 1. Запустить XMPP сервер

```bash
spade run
```

### 2. Запустить систему (в другом терминале)

```bash
source venv/bin/activate
python main.py
```

## Результат

Система вычисляет среднее и выводит:
- Структуру дерева
- Процесс агрегации
- Итоговое среднее значение
- Стоимость операций для каждого агента
- Общую стоимость алгоритма

## Пример вывода

```
Вычисленное среднее: 50.6667
Ожидаемое среднее:   50.6667

ОБЩАЯ СТОИМОСТЬ: 1005.99
```
